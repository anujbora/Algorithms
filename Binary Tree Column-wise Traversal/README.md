Given a binary tree, it traverses the tree in vertical manner.

For example, given following binary tree :

    	   1
        /    \
       2      3
      / \    / \
     4   5  6   7

A solution set is:
[
  [4],
  [2],
  [1, 5, 6],
  [3],
  [7]
]
